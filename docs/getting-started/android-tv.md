# Android TV / Google TV

:::caution Planned
Android TV support via Device Code Flow (RFC 8628) is planned but not yet implemented.
:::

The SDK will support TV platforms using the OAuth 2.0 Device Authorization Grant, which allows authentication on devices with limited input capabilities.

Check back for updates or follow the [main repository](https://github.com/swisstxt/srg-login-mobile-sdk) for progress.
