AccessToken

data class AccessToken(val value: String, val expiresAt: Long, val issuer: String?, val subject: String?, val issuedAt: Long?)

Access token with parsed JWT claims.

This type provides a type-safe wrapper around the raw JWT token string, exposing parsed claims directly for easy access without manual JWT parsing.

expiresIn and isExpired are computed properties that reflect the current system clock — they are NOT frozen at construction. This protects consumers from the post-persistence staleness bug: a token deserialized from storage and read 10 minutes later returns the correct (smaller) expiresIn. For TV / set-top devices that may have unsynchronized clocks, prefer isExpiredWithSkew with a tolerance.
