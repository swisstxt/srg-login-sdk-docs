FrontChannel

data class FrontChannel(val authContext: PlatformAuthContext, val postLogoutRedirectUri: String? = null, val onReturn: (String) -> Unit? = null) : LogoutType

Front-channel logout - redirects user to the authorization server's logout endpoint via browser to terminate the server session. Also clears local tokens and session data.

Uses fire-and-forget approach: returns success immediately after launching browser. Optional callback can be used for UX improvements when user returns via deeplink.
