# Web / JavaScript

:::caution Planned
Web/JS support is planned but not yet implemented. The Kotlin/JS target is currently disabled.
:::

The SDK will provide a JavaScript/WASM module for web applications, following the same API patterns as the Android and iOS SDKs.

Check back for updates or follow the [main repository](https://github.com/swisstxt/srg-login-mobile-sdk) for progress.
